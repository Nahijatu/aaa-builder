# AI Agents

#andreas-course #agent #automation

## Ringkasan

Agent adalah upgrade dari [[smart-tools|Smart Tools]]: nambahin memory loop + feedback system, jadi sistemnya bisa belajar dari pengalaman dan improve output-nya sendiri — bukan cuma jalanin tugas repetitif secara kaku.

## Poin Kunci

- Tools sifatnya kaku, cuma jalanin tugas repetitif tertentu, **tanpa proses pembelajaran** (<Course 5, [00:02:22](https://www.youtube.com/watch?v=1k75BoBBDc4&t=142s)>)
- Agent = tools inti + **memory loop** + **feedback system** — 3 komponen inti yang bikin Agent bisa "inget" masa lalu & evaluasi performanya sendiri (<Course 5, [00:06:20](https://www.youtube.com/watch?v=1k75BoBBDc4&t=380s)>)
- Kapan butuh Agent vs cukup Tools: kalau tugasnya repetitif & statis → [[smart-tools|Tools]] udah cukup; kalau butuh dinamis (misal niru gaya bahasa seseorang) → butuh Agent (<Course 5, [00:11:17](https://www.youtube.com/watch?v=1k75BoBBDc4&t=677s)>)
- Contoh: **Cruise** (Script Writer, awalnya cuma Tools di [[connecting-smart-tools|Course 4]]) diupgrade jadi Agent supaya bisa generate script yang dinamis & sesuai gaya bahasa orang (<Course 5, [00:17:28](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1048s)>)
- Contoh: **Ken** — Agent yang bisa baca link dari Twitter/TikTok/YouTube/blog, PDF, dan image; punya sistem rating 1-4; konten rating 4 di-lock & disimpan ke memori Ken (<Course 5, [00:23:07](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1387s), [00:24:02](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1442s)>)
- Kalau ada pertanyaan soal Agent, bisa tanya ke **Ask Andre Bot** (<Course 5, [00:25:23](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1523s)>)
