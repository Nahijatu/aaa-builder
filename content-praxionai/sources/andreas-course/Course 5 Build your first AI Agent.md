Concise summary


## Pengantar tentang Tools dan Agent
- Tools yang dibuat sebelumnya masih memiliki sifat yang kaku dan hanya menjalankan tugas tertentu secara repetitif, tanpa proses pembelajaran [(00:02:22)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=142s).
- Untuk membuat tools menjadi agents, perlu ditambahkan memori loop dan feedback system, sehingga tools dapat mengingat pengalaman dan memperbaiki diri sendiri [(00:05:27)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=327s).
- Agents yang dibangun harus memiliki tiga komponen, yaitu tools inti, memori loop, dan feedback system, sehingga dapat mengingat masa lalu dan mengevaluasi performanya [(00:06:20)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=380s).
- Untuk membuat AI Agent, perlu mempertimbangkan apakah tugasnya repetitif atau tidak, karena jika repetitif, tools saja sudah cukup, tetapi jika memerlukan dinamis dan niru suara, maka perlu menjadi agent [(00:11:17)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=677s).

## Komponen dan Fungsi Agent
- Contoh tools yang dapat diupgrade menjadi agent adalah script drafting seperti Ken, yang harus dinamis dan menyerupai gaya bahasa seseorang [(00:13:13)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=793s).
- Untuk membuat agent, perlu menambahkan memory loop dan feedback system, sehingga agent dapat belajar dari pengalaman dan meningkatkan kualitas outputnya [(00:16:25)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=985s).
- Contoh agent yang dapat dibuat adalah Cruise, yang dapat menggenerate script dengan kualitas yang sesuai dengan keinginan seseorang dan gaya bahasa yang menyerupai seseorang [(00:17:28)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1048s).

## Contoh dan Fungsi Ken sebagai Agent
- Ken dapat membaca link dari berbagai platform seperti Twitter, TikTok, YouTube, dan blog, serta dapat membaca PDF dan image [(00:23:07)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1387s)
- Sistem rating telah dibuat untuk menilai konten, dengan skor 1-4, dan skor 4 akan membuat konten tersebut di-lock dan disimpan dalam memori Ken [(00:24:02)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1442s)
- Ken memiliki memori loop dan sistem feedback, sehingga dapat menjadi agent yang lebih pintar dan dapat diarahkan untuk melakukan tugas-tugas tertentu [(00:24:51)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1491s)
- Pembuat konten dapat menggunakan Ken untuk membuat script dan konten lainnya, dan dapat menghubungi Ask Andre Bot jika memiliki pertanyaan atau memerlukan bantuan [(00:25:23)](https://www.youtube.com/watch?v=1k75BoBBDc4&t=1523s)
