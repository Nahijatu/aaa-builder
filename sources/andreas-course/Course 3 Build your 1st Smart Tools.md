## Proses Pengembangan dan Pengujian Smart Tools
- Proses membuat smart tools dimulai dengan mencari dan mengirimkan ID, dan setelah itu dilanjutkan dengan membuat API key dan melakukan quick test [(00:27:33)](https://www.youtube.com/watch?v=OAKONJR-UkU&t=1653s)

## Pelaksanaan dan Hasil Akhir Smart Tools
- Setelah semua proses selesai, smart tools dapat dijalankan secara otomatis dan akan mengirimkan berita atau notifikasi sesuai dengan parameter yang telah ditentukan [(00:46:33)](https://www.youtube.com/watch?v=OAKONJR-UkU&t=2793s)
